## Save the Campers (팀명 : SmartCampers)



# 1. 프로젝트 개요
### 주제명 : IoT기반 캠핑안전관리시스템 
- 일산화탄소·이산화탄소 감지 센서, 온·습도 센서 등을 활용하여 상황 발생 시 사용자에게 스피커로 전달
- 캠핑장 관리자 전용 페이지
- 캠핑장 이용자 전용 모바일 웹



# 2. 주요기술
- 다양한 IoT 센서를 활용하여 캠핑 이용객들에게 안전 관리 서비스를 제공함.
